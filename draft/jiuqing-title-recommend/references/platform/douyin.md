# 抖音平台规范

适用于目标平台为「抖音」的标题生成。

## 平台特征

- **核心特征**：直接冲击，强情绪/强对比，前 3 秒定生死
- **标题字数**：15–22 字（短视频信息流中标题显示极短）
- **读者行为**：无目的刷，标题+封面共同决定点击，情绪先于理性
- **调性**：直接、有冲击力、不说废话。长句和复杂逻辑在这里是劣势

## 评分调整

| 条件 | 调整 |
|------|------|
| 标题 > 22 字 | 清晰度 -1（信息流截断） |
| 含强情绪词（"震惊""直接""千万别"） | 点击理由 +1（上限 5） |
| 含数字对比/时间对比 | 具体性 +1（上限 5） |
| 含复杂逻辑/长定语 | 清晰度 -1（阅读成本高） |

## 专属公式

### 1. 强烈对比式
模板：「[之前] vs [之后]，差距太大了」
示例：「用AI写文案前 vs 用AI写文案后，差距太大了」

### 2. 情绪引爆式
模板：「看完这个 [内容类型]，我直接 [强烈反应]」
示例：「看完这个视频，我直接把之前的方法全扔了」

### 3. 数字冲击式
模板：「[大数字] 人不知道的 [小数字] 个真相」
示例：「99%的人不知道的3个AI真相」

### 4. 警告式
模板：「千万别 [错误做法]！[代价]」
示例：「千万别再用AI直接生成文章了！后果比你想的严重」
