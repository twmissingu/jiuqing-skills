# jiuqing-title-craft

## 痛点
标题平庸、CTR低。读者扫一眼就划过，内容再好也白费。

## 输入 → 输出
**输入**：文章内容/主题 + (可选)目标平台  
**输出**：10–15 个标题候选，每个附评分，推荐 Top 3

## 流程

1. **调用 `idea-grill`**，对齐以下信息：
   - 目标平台（公众号 / 小红书 / B站 / 抖音 / 通用）
   - 目标受众画像
   - 内容类型（教程 / 观点 / 测评 / 故事 / 资讯）
   完成标志：拿到平台+受众+内容类型三要素，缺任一则追问用户。

2. **按 8 种公式生成标题**（每类 2–3 个，共 16–24 个，择优输出 10–15 个——8 公式覆盖主要标题套路，10-15 候选足够对比择优又不至于选择疲劳）：

| 公式 | 示例 |
|------|------|
| **对比型** | "A vs B，差距有多大" / "5年前用A，现在换成B，因为…" |
| **痛点型** | "为什么XX总是失败" / "XX做错这3件事，浪费了半年" |
| **结果型** | "我用XX，3个月做到了YY" / "月入5万后，我停止了做XX" |
| **揭秘型** | "XX背后的真相" / "XX不敢告诉你的3个秘密" |
| **清单型** | "5个让你XXX的方法" / "想XX？先看完这4条建议" |
| **反直觉型** | "XX竟然是错的" / "停止XX后，效率反而提升了" |
| **数据型** | "XX%的人都不知道" / "我测试了10个工具，只有1个过关" |
| **提问型** | "XX真的值得买吗" / "为什么XX越来越难做" |

3. **评分标准**（每个标题三维度评分，各 1–5 分）：
   - **吸引力**：能否在 3 秒内激发点击欲（移动端平均注意力窗口约 3-5 秒）
   - **清晰度**：读者能否迅速理解文章主旨
   - **平台适配度**：是否符合目标平台标题规范

4. **推荐 Top 3**并逐条说明理由（每条理由引用对应标题的吸引力+清晰度得分作为依据）。

5. **按平台偏好调整**：

| 平台 | 策略 | 字数 |
|------|------|------|
| 公众号 | 信息密度高、包含核心关键词 | 25–30 字 |
| 小红书 | emoji + 短标题，口语化强 | 15–20 字 |
| B站 | 悬念/好奇心缺口，留钩子 | 20–25 字 |
| 抖音 | 直接冲击，强情绪/强对比 | 15–22 字 |

## 边界
- 不自动发布，标题仅为候选
- 不做 A/B 测试
- 最终由用户选择确认
- 不承诺效果（"10万+""爆款"等），标题质量取决于内容、时机、平台算法等不可控因素

## 失败路径

| 情况 | 信号 | 处理 |
|------|------|------|
| 用户只说"起个标题"，无主题/内容 | 输入缺少文章主题或内容描述 | 追问："请提供文章主题或内容概要，以及目标平台（公众号/小红书/B站/抖音）" |
| 目标平台不在已列平台中 | 用户指定知乎/Twitter/Newsletter等 | 按通用标准生成，注明"未收录该平台专属规范，按通用标题原则处理" |
| 生成的标题全部被用户否定 | 用户说"都不满意""再来一批" | 分析否定原因（太长/太短/不够吸引/偏离主题），调整公式权重后重新生成，最多迭代 3 轮 |
| 用户要求承诺效果 | "保证10万+""要爆款" | 明确拒绝："标题质量受内容、发布时间、平台算法等多因素影响，无法承诺具体数据。我能做的是按高CTR原则生成候选。" |
| idea-grill 调用失败或返回信息不足 | idea-grill 超时或返回空/模糊 | 跳过对齐，直接用用户原始输入生成标题，标注"未经对齐，基于原始输入生成" |
| 用户要求超出标题范围 | SEO标题/meta描述/关键词标签 | 拒绝并聚焦："本 skill 只生成文章标题候选，SEO相关请使用专门工具。" |

## 完整示例

**输入主题**：介绍一款新推出的 AI 笔记工具（功能：语音转文字、自动标签、跨平台同步）

**输出 3 个标题 + 评分**：

| 标题 | 吸引力 | 清晰度 | 平台适配 | 总分 |
|------|--------|--------|----------|------|
| 用了这AI笔记工具30天，我再也没打开过Notion | 5 | 4 | 4（B站/抖音） | 13 |
| 为什么你记的笔记永远不会看？3个错误+1个解法 | 5 | 5 | 5（公众号） | 15 |
| 我测试了15款笔记App，只有这1个能自动帮你分类 | 4 | 5 | 4（小红书） | 13 |

**Top 3 推荐理由**：
1. **🥇 "为什么你记的笔记永远不会看？3个错误+1个解法"** — 痛点直击（记了不看），清单结构清晰，公众号/通用皆可，吸引力与清晰度双高。
2. **🥈 "用了这AI笔记工具30天，我再也没打开过Notion"** — 对比强烈（vs Notion），结果具体（30天），适合B站/抖音做悬念开场。
3. **🥉 "我测试了15款笔记App，只有这1个能自动帮你分类"** — 数据可信（15款），突出核心卖点（自动分类），小红书适配度高（简短+数字）。
